欢迎阅读autoops_common模块接口文档
===================================

autoops_common模块主要包含dao, services, utils三个模块.结构如下:

* dao模块为sql语言
* util模块为数据库初始化代码
* services模块提供外部接口

因此dao和util未作描述, 请主要查看services模块提供的接口介绍.
